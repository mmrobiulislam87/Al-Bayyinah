# Al-Bayyinah

কুরআন অনুসন্ধান, রিসার্চ ও শেখার ওয়েব অ্যাপ্লিকেশন। **সীমা:** হাদিসের বর্ণনা, রেফারেন্স বা হাদিস ডাটা এই প্রজেক্টে অন্তর্ভুক্ত নয় — বিস্তারিত [`ROADMAP.md`](ROADMAP.md)।

| বিষয় | পথ |
|------|-----|
| Next.js অ্যাপ | [`web/`](web/) |
| রোডম্যাপ ও টাস্ক ট্র্যাকার | [`ROADMAP.md`](ROADMAP.md) — **কোরআন গবেষণা** বিভাগ ও ডাটা স্তর |
| গবেষণাগার মাস্টার প্ল্যান | [`docs/RESEARCH_LAB.md`](docs/RESEARCH_LAB.md) |

| ডাটা বিল্ড স্ক্রিপ্ট | [`scripts/`](scripts/) |
